Console Typer

by daniel zamorano

This is a funny util written using pure javascript to give a terminal typing effect to any paragraph. This is a sample paragraph using it.

Options
Attribute Option Description
loop boolean loop the effect after finish
loopAfterSeconds number seconds it waits before start restart the effect.(default: 6)
cursor string character to use as cursor (default: "I")
stopCursorAfterBlinks number Number of blinks that the cursor will do before stop. default (10)
cursorAnimationSpeedMs number Speed of cursor blikn, miliseconds: (default: 500)
typingSpeedMs number Typing effect speed (deafult: 70)
onStartTyping() function Executes after starting typing
onStopTyping function Executes after stop typing
onStopCursorAnimation function executes after cursor animation stops
